
Specs for Android app version of Obsyncian
1. The app should consist of a single screen that on first use requires the user to 
set a directory path to a config.json file that is used for setting. A file picker 
thatpops up and allows the user to search the device for the file would be perfect.
2. The app should handle not being able to sync when there is not internet connectivity, 
and sync once again once connectivity is restored.
3. The app should always run in the background, to sync when needed.
4. The app should be built with Flutter and use latest versions of Flutter, and Flutter packages.
5. document the specifications and architecture of this app in separate files under a docs/ directory
